# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-12-31

### Added

- Initial release
- Tree-sitter grammar for AppleScript
- Syntax highlighting for keywords, strings, numbers, operators, and more
- Support for `.applescript` and `.scpt` file extensions
- Line comments (`--`) and block comments (`(* *)`)
- Auto-indentation for block structures (handlers, tell blocks, if/repeat/try statements)
- Code outline showing handlers, properties, and tell blocks
- Bracket matching for parentheses, braces, and block keywords
- Vim-style text objects for handlers and blocks
